# **Voz del Este**

Voz del Este is a web application developed as an academic project to simulate the management system of an online radio station. The system was built using the MVC architecture and allows administrators and authorized users to manage radio programs, news, sponsors, and other dynamic content while providing real-time information such as weather and currency exchange rates.

The platform also implements a role-based access control system with dynamic permissions, allowing different types of users to access specific functionalities depending on their assigned permissions.

# **Main Features**

* User authentication and role-based authorization  
* Dynamic permissions management system  
* CRUD operations for programs, sponsors, news, users, and roles  
* Weekly radio program schedule management  
* Integration with external APIs for weather and currency exchange  
* Live transmission management module  
* Dashboard with dynamic information  
* Image storage within the database  
* Responsive and user-friendly interface

# **Technologies Used**

* ASP.NET MVC 5  
* Entity Framework (Database First)  
* SQL Server  
* HTML  
* CSS  
* Bootstrap  
* JavaScript  
* External APIs

# **Project Purpose**

This project was developed as the final project for the Programming course of the third semester in the *Analista Programador* program at **CEI – CLAEH**. The objective was to design and implement a complete web application following the MVC architecture, focusing on structured development, database management, role-based permissions, and integration with external services.
